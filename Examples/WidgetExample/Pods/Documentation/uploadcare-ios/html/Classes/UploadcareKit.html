uploadFileFromURL:progressBlock:successBlock:failureBlock:

Uploads a file from URL

- (void)uploadFileFromURL:(NSString *)url progressBlock:(UploadcareProgressBlock)progressBlock successBlock:(UploadcareSuccessBlock)successBlock failureBlock:(UploadcareFailureBlock)failureBlock

Parameters

url

The URL used to retrieve the file.

progressBlock

The block to call repeatedly during the upload. Receives two arguments: long long bytesDone and long long bytesTotal.

successBlock

The handler block to call when the upload is completed succesfully. Receives a single string argument fileId.

failureBlock

The handler block to call when the upload fails due to an error. Receives a single argument NSError *error

uploadFileWithName:data:contentType:progressBlock:successBlock:failureBlock:

Uploads an arbitrary file (e.g. an image, a movie clip, a spreadsheet document, etc.) with the content provided by the NSData argument.

- (void)uploadFileWithName:(NSString *)filename data:(NSData *)data contentType:(NSString *)contentType progressBlock:(UploadcareProgressBlock)progressBlock successBlock:(UploadcareSuccessBlock)successBlock failureBlock:(UploadcareFailureBlock)failureBlock

Parameters

filename

The file name to assign to the file uploaded.

data

The data to upload.

contentType

The Internet media type of the file. If nil, the type is auto-detected from the file name.

progressBlock

The block to call repeatedly during the upload. Receives two arguments: long long bytesDone and long long bytesTotal.

successBlock

The handler block to call when the upload is completed succesfully. Receives a single string argument fileId.

failureBlock

The handler block to call when the upload fails due to an error. Receives a single argument NSError *error
